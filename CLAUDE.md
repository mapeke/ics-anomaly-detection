## Datasets

1. **HAI 21.03** — `data/raw/hai/hai-21.03/` — ETRI HIL-based ICS dataset.
   Gzipped CSVs (`hai-21.03/*.csv.gz`), 79 sensor channels, per-process attack
   flags (`attack_P1/P2/P3`) consolidated into a single `label`. Cloned from
   https://github.com/icsdataset/hai (CC BY 4.0). **Defaulting to 21.03
   because 22.04/23.05 are Git-LFS-tracked and the upstream LFS budget is
   exhausted (smudge failures).** 20.07 and 21.03 are plain gz CSVs that
   check out normally.
2. **Morris Gas Pipeline** — `data/raw/morris/IanArffDataset.arff` —
   Mississippi State gas pipeline Modbus RTU dataset. ARFF format; parsed
   via custom `_parse_arff_header` + `pd.read_csv` (liac-arff is too strict
   for this file's malformed rows). 17 features after dropping
   label/metadata columns (`binary/categorized/specific result`) and the
   monotonic UNIX `time` column (would leak train/test split info).

Retrieval instructions live in `data/README.md`. `data/raw/` and
`data/processed/*.npz` are gitignored.

## Models

Four models compared across both datasets:

| Model | File | Best result |
|---|---|---|
| Isolation Forest | `src/models/isolation_forest.py` | F1=0.73 on Morris |
| One-Class SVM | `src/models/ocsvm.py` | F1=0.65 on Morris |
| Dense Autoencoder | `src/models/autoencoder.py` | F1=0.74 Morris, F1=0.52 HAI |
| LSTM Autoencoder | `src/models/lstm_autoencoder.py` | weak on both |

All trained unsupervised on normal data only. Thresholds are tuned on a
small random slice of the test set (oracle-threshold protocol) — a known
limitation noted in the thesis discussion.

## How to run

```bash
pip install -r requirements.txt

# Fetch datasets (one-time)
cd data/raw
git clone --depth 1 https://github.com/icsdataset/hai.git hai
cd hai && git lfs install --local   # harmless; 21.03 isn't LFS
cd ../
mkdir morris && cd morris
curl -O http://www.ece.uah.edu/~thm0009/icsdatasets/IanArffDataset.arff

# Run notebooks end-to-end

```

## Known gotchas (from this project)

- **TensorFlow install on Windows**: pip may report success but leave
  `tensorflow_intel` incomplete (top-level `tensorflow/` directory with no
  `__init__.py`). Fix: `pip install --force-reinstall --no-deps tensorflow==2.17.1 tensorflow-intel==2.17.1`.
- **Keras API**: use `from tensorflow.random import set_seed` (not
  `tf.random.set_seed` — `tf.random` attribute isn't exposed before a full
  sub-module import on some TF 2.17 builds).
- **`tabulate`** is required by `pandas.DataFrame.to_markdown()` in
  notebook 05 — included in `requirements.txt`.
- **Morris LSTM windowing** produces a degenerate all-attack test set at
  the current stride — documented in notebook 05; for a thesis rerun either
  reduce stride, re-shuffle normal packets, or treat LSTM as HAI-only.
